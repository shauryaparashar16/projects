# Advance Keylogger

A Python-based keylogging application with a GUI interface built using Tkinter.  
The project demonstrates keyboard event monitoring, local logging, and optional cloud synchronization using Firebase.
## Project Structure

AdvanceKeylogger/
│
├── kelogg.py       # Local keylogger with GUI
├── keyrem.py       # Firebase remote keylogger
├── kelogg.pyw      # Windows background version (no console)
├── key_log.txt     # Generated log file
└── img.jpg         # GUI background image
## Installation

Clone the repository:

git clone https://github.com/shauryaparashar16/projects.git

Navigate to the project directory:

cd projects/AdvanceKeylogger

Install required dependencies:

pip install pynput Pillow firebase-admin
## Technologies Used

- Python
- Tkinter (GUI)
- pynput (Keyboard monitoring)
- Pillow (Image handling)
- Firebase Realtime Database
